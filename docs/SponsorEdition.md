# FolderRewind Sponsor Edition Policy

FolderRewind is a free and open-source backup tool. Sponsor Edition is an optional Microsoft Store durable add-on for users who want to support ongoing development and receive small cosmetic perks.

It's free for FolderRewind source code repository contributors and those who offer thought-provoking feedback or support.

## What Sponsor Edition Unlocks

Sponsor Edition unlocks appearance-only options:

- Custom app accent color presets.
- Switching the main window material between Mica and Acrylic.
- Custom title bar text.
- Custom title bar glyph from the built-in WinUI icon set.
- Custom app background image with fit, opacity, and overlay controls.
- Custom audio file for the backup and restore completion sound.

These perks are intentionally lightweight. Core backup, restore, history, automation, plugin, and template features remain available in the free version.

## Purchase and Promo Codes

- The paid Sponsor Edition add-on is sold through Microsoft Store as a durable add-on.
- After purchase, FolderRewind checks the Microsoft Store add-on license and unlocks the cosmetic options.
- Contributors and third-party sponsors may receive Microsoft Store promo codes from the maintainer.
- Promo codes are redeemed through Microsoft Store or the Microsoft account redemption flow. FolderRewind does not provide an in-app promo-code input box.

## Offline Behavior

Sponsor Edition relies on Microsoft Store licensing. A purchased add-on is expected to be recognized through the local Store license cache when available, but initial purchase, promo-code redemption, and account synchronization require Microsoft Store access.

## Open Source Expectations

FolderRewind remains open source. Because the client code is public, Sponsor Edition is not intended as a hard DRM boundary. The goal is to provide a clear, convenient way for users to support development without reducing the free version.

## Privacy Notes

FolderRewind checks Microsoft Store licensing locally and does not intentionally write detailed add-on identifiers, Store collection entries, order details, Microsoft account data, or promo-code information to application logs.

## Refunds and Support

Purchases and refunds are handled by Microsoft Store according to Microsoft Store policies. If the add-on is purchased but FolderRewind does not recognize it, try:

1. Sign in to Microsoft Store with the purchasing account.
2. Restart FolderRewind.
3. Open Settings, then use Restore purchase or Refresh status.
4. If it still fails, open an issue with your FolderRewind version, Windows version, and Store purchase status. Do not post private order details publicly.
